#  Cadastro de Clientes - AUTI LLC

##  Sobre o Projeto

O 'Cadastro de Clientes - AUTI LLC' foi desenvolvido em Python para auxiliar empresas de varejo no controle e organização das informações de seus clientes.

O sistema permite cadastrar, visualizar, editar e excluir dados de clientes através de uma interface gráfica simples e intuitiva.

Ele foi desenvolvido utilizando 'Python, Tkinter e SQLite', proporcionando um sistema leve, rápido e fácil de utilizar.

---

# Objetivo

O objetivo principal do sistema é organizar o cadastro de clientes da AUTI LLC, resolvendo problemas como:

- dificuldade em localizar informações de clientes
- falta de organização dos dados
- ausência de um sistema simples de gerenciamento

Com o sistema é possível gerenciar os dados de forma rápida e segura.

---

# Requisitos do Sistema (Stack utilizada)

Para executar o sistema é necessário:

### Linguagem
- Python 3.x

### Bibliotecas utilizadas
- tkinter → interface gráfica
- sqlite3 → banco de dados local
- ttk → estilização da interface

### Banco de Dados
- SQLite

### Ambiente recomendado
- VS Code ou PyCharm

---

#  Funcionalidades do Sistema

O sistema possui um CRUD completo de clientes:

Cadastro de novos clientes  
Visualização de clientes cadastrados  
Atualização de dados  
Exclusão de clientes  

### Informações cadastradas

- Nome
- Email
- Telefone
- Endereço
